    MuditaKurye Third-Party Integration API¶

    MuditaKurye platformuna harici sistemlerden sipariş gönderimi ve durum bildirimleri için API dokümantasyonuna hoş geldiniz.

    Hızlı Başlangıç¶

    Bu dokümantasyon, POS entegratörlerinin MuditaKurye platformu ile nasıl entegre olacağını açıklar:

    • Sipariş Gönderimi: REST API üzerinden yeni siparişler oluşturun
    • Durum Güncellemeleri: Webhook'lar aracılığıyla gerçek zamanlı sipariş durumu bildirimleri alın
    • Kimlik Doğrulama: API Key veya Basic Auth ile güvenli erişim

    API Base URL¶

    Ortam URL
    Production https://api.muditakurye.com.tr
    Staging https://staging-api.muditakurye.com

    HTTPS Zorunludur

    Tüm istekler HTTPS üzerinden yapılmalıdır. HTTP bağlantıları desteklenmez.

    Kimlik Bilgilerinizi Nasıl Alırsınız?¶

    1. MuditaKurye restoran yönetim paneline giriş yapın
    2. Restoran ayarlarına gidin
    3. "Entegrasyon" sekmesini açın
    4. API Key veya kullanıcı adı/şifre oluşturun

    Önemli Güvenlik Notu

    Kimlik bilgileri sadece bir kez gösterilir ve tekrar alınamaz. Güvenli bir yerde saklayın.
